Головоломки - доступные мини-игры для китов

Пятнашки

Описание: классическая игра, выдающая после прохождения код для следующего задания. Результат, полученный до этого, влияет на количество пятнашек в наборе

Возможность задать:

  • Тип ключа
  • Сложность

Виселица

Описание: нужно угадать слово определённой длины, выбирая по одной букве

Возможность задать:

  • Тип ключа
  • Сложность
  • Вопрос и ответ

Шифрование перестановкой

Описание: дан ключ из набора символов. Разгадайте с помощью него слово или предложение заданной длины, зашифрованное методом перестановки

Возможность задать:

  • Тип ключа
  • Сложность
